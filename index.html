B-Tree Algorithm

The goal of this project was to implement a "B-Tree" algorithm and, additionally, measure how much we can improve that structure using some amount of cache memory.

Accomplishments
  • Tools: C++, Thomas H. Cormen book on algorithms
  • B-Tree is self-balancing tree data structure that maintains sorted data and enables searching, sequential access, insertions and deletions in logarithmic time.
  • Because the cache size is much smaller than the B-Tree size, you can store only selected entries. The policy to be implemented is FIFO.
  • A measure of cache impact is the number of disk accesses (measured as B-Tree depth).
